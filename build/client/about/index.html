Webapp template Note to self on how to do this

This is just a template project for getting up and running quickly, with examples on most common scenarios, with these techs(MEAN stack):

  • Node.js/Express
  • MongoDb(using mongoose models exposed through Mers, consumed by restangular on the client side)
  • Angular with routing, etc already set up
  • Bootstrap 3
  • Passport for authentication
  • Various components for common tasks(e.g. bootstrap date picker)
